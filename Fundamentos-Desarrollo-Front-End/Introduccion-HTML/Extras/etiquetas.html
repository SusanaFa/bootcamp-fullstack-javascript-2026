Documento

<!doctype html>
Declara HTML moderno (modo estándar en el navegador).
<body>
Contenido visible del documento.
<html>
Elemento raíz del documento.

Metadata

<base>
Define URL base para enlaces/recursos relativos.
<head>
Contenedor de metadata: títulos, scripts, estilos, meta tags.
<link>
Vincula recursos externos (CSS, iconos, preloads, etc.).
<meta>
Metadata (charset, viewport, SEO, etc.).
<style>
CSS embebido.
<title>
Título de la pestaña/ventana.

Seccionamiento (estructura del contenido)

<address>
Información de contacto del autor/organización (del contenido cercano).
<article>
Contenido autocontenido (post, noticia, tarjeta completa, etc.).
<aside>
Contenido tangencial (sidebar, notas, relacionados).
<footer>
Pie de página o de sección.
<header>
Encabezado de página o sección.
<h1>…<h6>
Títulos jerárquicos.
<hgroup>
Agrupa un título con subtítulo(s) como unidad.
<main>
Contenido principal único de la página.
<nav>
Bloque de navegación.
<section>
Sección temática del documento.

Contenido de texto (bloques)

<blockquote>
Cita en bloque.
<dd>
Definición/detalle en lista de descripción.
<div>
Contenedor genérico (sin semántica propia).
<dl>
Lista de descripción (términos y definiciones).
<dt>
Término en lista de descripción.
<figure>
Contenido auto contenido (imagen, diagrama, snippet) con posible leyenda.
<figcaption>
Leyenda de <figure>.
<hr>
Separador temático (cambio de tema/escena).
<li>
Ítem de lista.
<menu>
Lista de comandos o acciones (similar a <ul> en muchos casos).
<ol>
Lista ordenada.
<p>
Párrafo.
<pre>
Texto preformateado (preserva espacios y saltos).
<ul>
Lista no ordenada.

Semántica inline (dentro del texto)

<a>
Enlace (navegación o ancla).
<abbr>
Abreviatura (ideal con title explicativo).
<b>
Resaltado estilístico sin énfasis semántico (úsalo con criterio).
<bdi>
Aísla direccionalidad de texto (útil en idiomas RTL/LTR mezclados).
<bdo>
Fuerza dirección del texto (rtl/ltr).
<br>
Salto de línea dentro de un bloque.
<cite>
Título de obra citada (libro, película, artículo).
<code>
Fragmento de código.
<data>
Asocia un valor legible con un valor “máquina” (value).
<del>
Texto eliminado (edición).
<dfn>
Definición del término.
<em>
Énfasis (importante para lectura y accesibilidad).
<i>
Voz alternativa (términos, pensamientos, tecnicismos).
<ins>
Texto insertado (edición).
<kbd>
Entrada de teclado.
<mark>
Resaltado por relevancia (como “marcador”).
<q>
Cita corta en línea.
<rp>
Paréntesis de ruby como fallback.
<rt>
Anotación ruby (pronunciación).
<ruby>
Marca anotaciones ruby (muy usado en japonés/chino).
<s>
Texto “no relevante/ya no válido” (no es edición como <del>).
<samp>
Salida/ejemplo de un programa.
<small>
Notas, letra chica, aclaraciones.
<span>
Contenedor inline genérico.
<strong>
Importancia fuerte (semántica, no solo “negrita”).
<sub>
Subíndice.
<sup>
Superíndice.
<time>
Fecha/hora (ideal con datetime).
<u>
Anotación no textual (p. ej. “nombre propio” en algunos estilos) — no “subrayar por subrayar”.
<var>
Variable o placeholder.
<wbr>
Oportunidad de quiebre de palabra (word break opportunity).

Multimedia

<audio>
Audio embebido.
<img>
Imagen (usa alt siempre).
<picture>
Imágenes responsivas (con <source>).
<source>
Fuente alternativa para audio/video/picture.
<track>
Subtítulos/captions para <video> o <audio>.
<video>
Video embebido.

Embebidos / gráficos

<canvas>
Superficie de dibujo (JS).
<embed>
Contenido externo embebido (con cuidado).
<iframe>
Incrusta otra página/documento.
<map>
Mapa de imagen (con <area>).
<object>
Recurso externo embebido (con fallback interno).
<param>
Obsoleta (no usar): se lista en “Obsoletas”.
<svg>
SVG inline (elemento de SVG; se usa dentro de HTML).

Tablas

<caption>
Título/leyenda de la tabla.
<col>
Define columna (usado con <colgroup>).
<colgroup>
Agrupa columnas.
<table>
Tabla.
<tbody>
Cuerpo de tabla.
<td>
Celda de datos.
<tfoot>
Pie de tabla.
<th>
Celda encabezado.
<thead>
Encabezado de tabla.
<tr>
Fila de tabla.

Formularios

<button>
Botón (submit/reset/button).
<datalist>
Lista de sugerencias para <input list="">.
<fieldset>
Agrupa controles relacionados.
<form>
Formulario.
<input>
Control de entrada (muchos type).
<label>
Etiqueta accesible para un control.
<legend>
Título del <fieldset>.
<meter>
Medición dentro de un rango conocido (no progreso).
<optgroup>
Agrupa opciones en <select>.
<option>
Opción en select/datalist.
<output>
Resultado calculado (puede asociarse a un form).
<progress>
Progreso de una tarea.
<select>
Selector de opciones.
<textarea>
Entrada de texto multilínea.

Interactivos / UI

<details>
Contenido desplegable.
<dialog>
Diálogo/modal nativo.
<summary>
Encabezado clickable de <details>.
<search>
Contenedor semántico para UI de búsqueda/filtrado (similar a landmark “search”).

Scripting

<noscript>
Contenido alternativo si JS está deshabilitado.
<script>
JavaScript embebido o externo.
<template>
Fragmento HTML no renderizado, útil para clonar con JS/Web Components.

Obsoletas (no usar)

Estas etiquetas existen por compatibilidad histórica, pero el estándar las marca como obsoletas/no conformes.

<acronym>
Obsoleta. Usa <abbr>.
<applet>
Obsoleta. Usa <embed> o <object>.
<basefont>
Obsoleta. Usa CSS.
<bgsound>
Obsoleta. Usa <audio>.
<big>
Obsoleta. Usa CSS o una etiqueta semántica adecuada (p. ej. <strong> o headings).
<blink>
Obsoleta. Usa CSS animations (si realmente lo necesitas… con misericordia 😅).
<center>
Obsoleta. Usa CSS (text-align, flex, grid).
<dir>
Obsoleta. Usa <ul>.
<font>
Obsoleta. Usa CSS.
<frame>
Obsoleta. Usa <iframe> o composición moderna.
<frameset>
Obsoleta. Usa CSS/layout moderno.
<isindex>
Obsoleta. Usa <form> + <input>.
<keygen>
Obsoleta. Hoy se usa Web Crypto + flujos modernos (según caso).
<listing>
Obsoleta. Usa <pre><code>.
<marquee>
Obsoleta. Usa CSS animations/transitions.
<menuitem>
Obsoleta. Menú contextual custom con JS.
<multicol>
Obsoleta. Usa CSS columns.
<nextid>
Obsoleta. Usa IDs/UUIDs.
<noembed>
Obsoleta. Usa <object> con fallback.
<noframes>
Obsoleta. Ya no aplica a web moderna.
<nobr>
Obsoleta. Usa CSS (white-space: nowrap).
<param>
Obsoleta. Usa <object data="..."> y atributos adecuados.
<plaintext>
Obsoleta. Se resuelve por MIME (text/plain), no por HTML.
<rb>
Obsoleta. Ruby moderno con <ruby>/<rt>/<rp>.
<rtc>
Obsoleta. Ruby moderno con <ruby> anidado si hace falta.
<spacer>
Obsoleta. Usa CSS (margin/padding/gap).
<strike>
Obsoleta. Usa <del> (edición) o <s> (no relevante).
<tt>
Obsoleta. Usa <code>, <kbd>, <samp> o CSS según intención.
<xmp>
Obsoleta. Usa <pre><code>.

Referencia: lista de elementos obsoletos del estándar HTML (WHATWG).

Tip: si lo usas como “chuleta”, lo ideal es abrirlo en el navegador y usar Ctrl/Cmd+F para buscar la etiqueta.
