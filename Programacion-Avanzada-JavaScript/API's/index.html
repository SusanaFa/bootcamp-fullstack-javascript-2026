Ejercicios de Consumo de APIs con JavaScript

Ejemplos desde lo más básico hasta casos un poco más completos, utilizando Fetch, JSON, manejo de errores y render al DOM.

1) Fetch básico: traer un post

Objetivo: comprender el flujo fetch → response → json() → data

2) Listar varios posts

Objetivo: recorrer un arreglo de datos recibido desde una API

3) Manejo de errores

Objetivo: validar response.ok y controlar errores

4) The Cat API

Objetivo: consumir una API externa y mostrar una imagen

5) PokéAPI

Objetivo: buscar un Pokémon según el nombre ingresado por el usuario

6) Async/Await con usuarios

Objetivo: consumir una API usando async/await en lugar de then/catch

7) Ver JSON en pantalla

Objetivo: observar la estructura real del objeto antes de trabajarlo
