| Archivo                  | Para qué sirve                                                   |
| ------------------------ | ---------------------------------------------------------------- |
| `ApiError.js`            | Crear errores controlados con código HTTP.                       |
| `asyncHandler.js`        | Evitar repetir `try/catch` en todos los controllers.             |
| `response.utils.js`      | Responder siempre con el mismo formato JSON.                     |
| `password.utils.js`      | Hashear y comparar contraseñas con bcrypt.                       |
| `jwt.utils.js`           | Crear y verificar tokens JWT.                                    |
| `uuid.utils.js`          | Validar IDs tipo UUID.                                           |
| `file.utils.js`          | Validar extensiones, generar nombres únicos y eliminar archivos. |
| `error.middleware.js`    | Centralizar errores.                                             |
| `auth.middleware.js`     | Proteger rutas privadas con JWT.                                 |
| `validate.middleware.js` | Ejecutar validaciones antes del controller.                      |
